# TravelSystem

基于Django框架的旅游网站，毕业设计、课程设计、网站设计。<font color="red">联系微信：YCK2050</font>

### 功能简介

- 旅游功能：景点介绍、景点列表、景点搜索、收藏景点、标记景点、旅行团介绍、旅行团列表、收藏旅行团、旅游团下单等功能。
- 旅游攻略功能：发布攻略，攻略分类、攻略标签、攻略搜索、数据可视化等功能。（支持富文本编辑）
- 用户功能：登录、注册、创建个人信息、修改个人信息、密码修改。
- 论坛功能：发布帖子、关注贴主、帖子点赞、点踩等功能。
- 评论功能：针对旅游攻略和论坛部分，可对文章和帖子进行评论，并且实现跟帖功能。

### 技术栈

- Python=3.10
- Django=4.2.6
- Bootstrap=5.2
- Jquery=3.6
